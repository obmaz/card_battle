# Dice Arena Codex Development Document

## Overview
This is a lightweight mobile companion app for a dice-based board game.

## Features
- Character selection
- Skill selection (choose 2 out of 5)
- Build confirmation
- Ready summary screen

## Flow
Home -> CharacterSelect -> SkillSelect -> BuildConfirm -> ReadySummary

## Notes
- App does NOT handle combat
- App is only for setup
